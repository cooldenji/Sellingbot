# Zed Zoee Bot — Telegram Store Bot

## Setup on Render.com
1. Push to GitHub
2. Render → New → Web Service
3. Connect repo
4. Set env vars: USER_BOT_TOKEN, ADMIN_BOT_TOKEN, ADMIN_PASSWORD
5. Build: npm install | Start: node index.js

## Files
- index.js — Entry point
- userBot.js — Customer bot
- adminBot.js — Admin panel
- database.js — SQLite DB
- utils.js — Helper functions
- config.js — Configuration
